```typescript

  { field: 'name', editable: true, editor: { type: 'text' } }
];
```
→ Full guide: `docs/features/CELL_EDITING.md`

**"How do I handle server-side data?"**
```typescript
gridDefs = {
  height: 600,
  dataStream: 'server-side',
  recordsTotal: 10000
};

onDataParamsChange(params: any) {
  // params: { filterConfig, sortBy, orderBy, recordsFrom, recordsTo }
  this.fetchFromServer(params).subscribe(data => {
    this.records = data;
  });
}
```
→ Full guide: `docs/api/GRID_CONFIGURATION.md`

## 🎨 Architecture Overview

### Design Pattern
- **Component-based** with Service Layer
- **State Management**: Local component state + Services
- **Change Detection**: OnPush (optimized)
- **Styling**: SCSS + CSS Custom Properties (themeable)

### Key Services

| Service | Purpose | Key Methods |
|---------|---------|-------------|
| `ColumnSizingService` | Column width management | `processColumnOptions()`, `reCalcColumnWidth()` |
| `DataManagerService` | Data operations | `canSelectAll()` |
| `FilterManagerService` | Filtering logic | `buildQueryModel()`, `filterData()` |
| `PageManagerService` | Pagination logic | `resetPagination()` |
| `ElementManagerService` | DOM references | `registerDatagridEl()` |

### Component Hierarchy
```
IvoryPresentableComponent (Main)
├── PresentableColumnControlsComponent
├── PresentableColumnHeaderFilterComponent
│   ├── PresentableTextFilterComponent
│   └── PresentableOptionsFilterComponent
├── PresentableRowComponent
│   └── PresentableCellEditorComponent
├── PresentableColumnResizerComponent
└── PresentablePaginatorComponent
```

## 🎯 API Reference (Quick)

### Main Component: `IvoryPresentableComponent`

**Inputs:**
```typescript
@Input() gridDefs: GridConfiguration;           // Grid config
@Input() columnDefs: PresentableColumn[];       // Column definitions
@Input() records: any[];                        // Data array
@Input() pagination: boolean = false;           // Enable pagination
@Input() recordsPerPage: number = 0;            // Records per page
@Input() recordsPerPageOptions: number[];       // Page size options
@Input() recordSelection: boolean = false;      // Enable row selection
@Input() cellEditing: boolean = false;          // Enable cell editing
@Input() columnControls: boolean = false;       // Show column controls
```

**Outputs:**
```typescript
@Output() dataparams: EventEmitter<any>;        // Server-side params
@Output() recordsSelected: EventEmitter<any[]>; // Selected rows
@Output() cellEdit: EventEmitter<CellEditEvent>; // Cell edit events
```

### Key Interfaces

**PresentableColumn:**
```typescript
interface PresentableColumn {
  field?: string;                    // Data field name
  title?: string;                    // Column header text
  width?: number | string;           // Column width
  minWidth?: number | string;        // Minimum width
  maxWidth?: number | string;        // Maximum width
  visible?: boolean;                 // Column visibility
  alignment?: string;                // Text alignment
  sortable?: boolean;                // Enable sorting
  resizable?: boolean;               // Enable resizing
  hasFilter?: boolean;               // Show filter
  filterType?: 'text' | 'options';   // Filter type
  filterOptions?: any[];             // Filter options
  editable?: boolean;                // Enable editing
  editor?: CellEditorConfig;         // Editor config
  pinned?: 'left' | 'right' | null;  // Pin position ⭐ NEW
}
```

**GridConfiguration:**
```typescript
interface GridConfiguration {
  height: number;                    // Grid height (px)
  dataStream: 'client-side' | 'server-side';
  recordsTotal?: number;             // Total records (server-side)
  style?: string;                    // Reserved
}
```

**CellEditorConfig:**
```typescript
interface CellEditorConfig {
  type: 'text' | 'number' | 'date' | 'select';
  min?: number;                      // For number type
  max?: number;                      // For number type
  options?: string[];                // For select type
}
```

**CellEditEvent:**
```typescript
interface CellEditEvent {
  rowData: any;                      // Complete row data
  field: string;                     // Edited field
  oldValue: any;                     // Previous value
  newValue: any;                     // New value
  rowIndex: number;                  // Row index
}
```

## 📝 Code Conventions

### Naming Conventions
- **Components**: `PresentableXxxComponent`
- **Services**: `XxxService`
- **Interfaces**: `PresentableXxx` or `XxxConfig`
- **CSS Classes**: `ivpt-xxx-yyy` (kebab-case)
- **CSS Variables**: `--ivpt-xxx-yyy`

### TypeScript Guidelines
- Interfaces for all public APIs
- Strict typing enabled
- Avoid `any`, prefer `unknown` or specific types
- Use readonly for immutable data

### File Organization
- One component per file
- Co-locate templates and styles
- Group related components in folders
- Keep services in dedicated `services/` folder

## 📊 Performance Guidelines

### Data Size Recommendations
- **< 1,000 rows**: Excellent performance (client-side)
- **1,000 - 10,000 rows**: Good performance (client-side)
- **> 10,000 rows**: Use server-side mode
- **> 100,000 rows**: Virtual scrolling (planned v1.2.0)

### Optimization Strategies
- OnPush change detection used throughout
- TrackBy functions in all loops
- Efficient DOM manipulation
- CSS-based styling (no inline styles)
- Lazy loading for server-side data
